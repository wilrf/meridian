# Python Learning Track

A curriculum focused on **why** things work, not just how. Built for deep understanding with applications in ML/AI and finance.

---

## Phases

| # | Phase | Focus |
|---|-------|-------|
| 1 | Foundations | Variables, types, memory, control flow |
| 2 | Data Structures | Lists, dicts, functions, classes |
| 3 | Engineering & External World | Modules, venvs, testing, files, APIs |
| 4 | Data Analysis | NumPy, pandas, visualization |
| 5 | Machine Learning | Statistics, regression, scikit-learn |
| 6 | Capstones | Full projects in finance/ML |

---

## Lesson Format

Every lesson follows the **Why-First Method**:

1. **The Problem** — What need does this solve?
2. **The Metaphor** — Layman's mental model
3. **The Technical Reality** — What's happening under the hood
4. **The Code** — Syntax and examples
5. **The Exercise** — Hands-on practice
6. **The Gotchas** — Common mistakes

---

## How to Use

This curriculum is designed to be run in the **Learning App**.

1. Navigate to the **Foundations** module.
2. Read the lesson.
3. Run the code examples directly in the browser.
4. Complete the exercises to verify your understanding.
